# Instagram Api

## Open Endpoints

Open endpoints require no Authentication.

* [Devs]: `POST /devs` -- List all devs
* [Devs]: `GET /devs` -- Create one Dev
* [Dev]: `GET /devs/:devTo/likes` -- Add like for Dev
* [Dev]: `GET /devs/:devTo/dislikes` -- Add dislike for Dev
